# Ansible Web Stack Automation

## Overview
This project automates the installation and configuration of Nginx using Ansible roles on macOS (Homebrew environment).

## Features
- Role-based Ansible structure
- Idempotent playbooks
- Jinja2 templating
- Environment inventory support
- Service management

## Requirements
- macOS
- Ansible installed via Homebrew
- Homebrew installed

## How To Run

ansible-playbook site.yml

## Test

curl http://localhost:8080

## Future Improvements
- Linux server support
- Firewall automation
- CI/CD integration
- Network device automation
